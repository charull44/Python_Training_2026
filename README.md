# Python Basic Programs

This repository contains basic Python programs created for practice and academic learning.

## Topics Covered

* Data Types
* Complex Number Conversion
* Boolean Conversion
* Simple Interest Calculation
* Number Swapping
* Unit Conversion
* Basic Number Logic

## Folder Structure

### Basics

Programs demonstrating Python data types and conversions.

* data_types_demo.py
* complex_conversion.py
* boolean_conversion.py

### Arithmetic Programs

Programs performing mathematical calculations.

* simple_interest.py
* height_conversion.py

### Logic Programs

Programs using logical operations and number manipulation.

* swap_numbers.py
* reverse_last_three_digits.py

## How to Run

1. Install Python.
2. Open the folder in Visual Studio Code.
3. Run any program using terminal:

python filename.py

Example:

python simple_interest.py
